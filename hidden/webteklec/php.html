Hypertext Preprocessor (PHP)

• Rasmus Lerdorf unleashed the first version of PHP way back in 1994.

• Recursive acronym for "PHP: Hypertext Preprocessor", it is one of the simplest server-side language, it was primarily designed for web development.

Basic PHP Syntax

• A PHP script can be placed anywhere in the document.

• A PHP script starts with <?php and ends with ?>

• The default file extension for PHP files is ".php".

• A PHP file normally contains HTML tags, and some PHP scripting code.

• Classes, functions, class constructors, class methods, keyworkds and constructs are NOT sensitive in PHP. However, variables, constants, array keys, class properties and class constants are case-sensitive.

PHP Data Types

String Integer Float (also called double) Boolean Array Object NULL Resource

• Several Ways of Commenting in PHP

                
                    <?php 
                       // This is a single-line comment

                        # This is also a single-line comment

                        /*
                        This is a multiple-lines comment block
                        */

                        // You can also use comments to leave out parts of a code line
                        $sum = 9 /* + 5 */ + 97;
                        echo $sum;
                    ?>

Superglobals

PHP $GLOBALS
This variable is used to access global variables from anywhere in the PHP script.
                            
                     <?php 
                         $x = 100;
                         $y = 20; 
                            function addition() {
                             $GLOBALS['z'] = $GLOBALS['x'] - $GLOBALS['y'];
                            }
                            addition();
                            echo $z;
                    ?>
PHP $_SERVER
This variable holds the information about the headers, paths, and script locations.
                            
                     <?php 
                         echo $_SERVER['SERVER_NAME'];
                    ?>
PHP $_GET
This variable is used to collect forms sent in the URL.
                            
         <?php 
            echo $_GET['http://stackoverflow.com/questions/5884807/get-url-parameter-in-php'];
        ?>
PHP $_POST
This variable is used to handle form data and pass variables.
                            
                     <?php 
                         echo "Welcome ". $_POST['name']. " to your tape.";
                    ?>
PHP $_FILES
This variable will contain all the uploaded file information.
                            
                     <?php 
                         if ($_FILES["fileToUpload"]["size"] > 1000000) {
                         echo "Sorry, your file is too large.";
                         $uploadOk = 0;
                    ?>
PHP $_COOKIE
This variable is being set when the page loads.
                            
         <?php 
            if (!isset($_COOKIE['visited'])) {
                setcookie('visited', true, time() + 3600 * 72); // Save a cookie for 3 day
        ?>
PHP $_SESSION
This variable stores all the session variable values.
                
                <?php 
                    // starts the session
                    session_start();
                    // stores session data
                    $_SESSION["username"] = "Jamie";
                ?>
PHP $_REQUEST
Using this variable is quite harmful, because you can be vunerable to a cross-site script attack.
It contains the contents of $_GET, $_POST and $_COOKIE.
                
                <?php 
                    $name=$_REQUEST['SirJo'];  
                    echo $name; 
                ?>
PHP $_ENV
This variable is used to return the environment variables from the web server.
