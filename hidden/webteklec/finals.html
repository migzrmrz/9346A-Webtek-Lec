Java EE
Java EE is an open, standards-based development and deployment platform for creating distributed, transactional, reliable, secure, multi-tiered, web-based, server-centric, component-based enterprise applications.

Java EE Application Model

Java Programming Language, Java Virtual Machine (JVM)

Java EE Server

Java EE Web Application Model

º Collection of resources installed under a specific subset of the URL namespace of a web application server compliant with the Java EE Specification

Examples: Apache‟s Tomcat, Apache‟s Geronimo, Sun Microsystems‟ Glassfish, IBM‟s WebSphere, etc

Resources:

• Static resources: web pages, images, stylesheets, etc.

• Dynamic resources: servlets, JSPs

• Miscellaneous resources: business object classes, support libraries, etc.

Examples of BOM classes: Java Beans, EJB
